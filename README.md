## 🇬🇧 English

### Overview
This project is an arcade-style, fast-paced shooter game inspired by Geometry Wars. It features retro aesthetics, dynamic gameplay, and engaging mechanics designed for maximum fun and replayability.

### Key Features
- Fast-paced arcade action
- Retro-style geometric graphics
- Various enemy types and challenging waves
- Custom physics and collision detection

### Getting Started
To play or modify the game, open the respective source files in your preferred IDE and compile the game using standard build tools. Ensure your graphics libraries are correctly configured.

## 🇹🇷 Türkçe

### Genel Bakış
Bu proje, Geometry Wars oyunundan esinlenilerek geliştirilmiş, hızlı tempolu bir arcade atış oyunudur. Maksimum eğlence ve tekrar oynanabilirlik sağlamak amacıyla retro grafikler ve dinamik oyun mekanikleri sunar.

- Hızlı tempolu arcade aksiyonu
